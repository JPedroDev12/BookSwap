# 📚 BookSwap

**BookSwap** é uma plataforma web inspirada no funcionamento do Tinder, mas voltada para **troca e descoberta de livros entre leitores**.

A ideia do projeto é conectar pessoas que possuem livros que outras desejam, criando **matches entre usuários interessados em trocar livros**.

Além da troca, a plataforma também conta com **chat entre usuários, perfil personalizado e uma loja para compra de livros**.

---

# 🚀 Funcionalidades

## 🔄 Sistema de Swapping (Estilo Tinder)

A principal funcionalidade do site é o sistema de **swipe de livros**, inspirado em aplicativos de relacionamento.

Funciona da seguinte forma:

1. O usuário visualiza livros que outras pessoas possuem.
2. Pode **aceitar ou recusar** o interesse na troca.
3. Caso aconteça interesse mútuo:

   * Você possui um livro que a pessoa deseja.
   * A pessoa possui um livro que você deseja.

🎉 **Um Match acontece!**

Após o match, os usuários podem conversar através do **chat da plataforma** para combinar a troca.

---

## 💬 Chat entre Usuários

Quando um **match** acontece, os usuários podem conversar através de um **chat privado** dentro do site.

O chat permite:

* Comunicação entre leitores
* Combinar detalhes da troca
* Interagir diretamente na plataforma

---

## 🛒 Loja de Livros

O BookSwap também possui uma **loja integrada**, onde os usuários podem:

* Navegar por diversos livros
* Visualizar capas e informações
* Comprar livros diretamente pelo site

---

## 👤 Perfil do Usuário

Cada usuário possui um **perfil personalizado**, contendo:

* Descrição / Bio
* Informações pessoais
* Organização da biblioteca pessoal

Os livros podem ser organizados nas seguintes categorias:

* 📖 **Lidos**
* 📚 **Não lidos**
* 🔍 **Quero ler**
* ⏳ **Estou lendo**
* ⭐ **Gostei**
* ❌ **Não gostei**

---

## 🌗 Tema Claro e Escuro

O site possui suporte para dois temas:

* ☀️ **Modo Claro**
* 🌙 **Modo Escuro**

Permitindo uma experiência melhor de leitura e navegação.
